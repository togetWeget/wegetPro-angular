Pourquoi togetPro ?
The special state called void can apply to any animation. It applies when the element is not attached to a view, perhaps because it has not yet been added or because it has been removed. The void state is useful for defining enter and leave animations. For example the * => void transition applies when the element leaves the view, regardless of what state it was in before it left The special state called void can apply to any animation.
